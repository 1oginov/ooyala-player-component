## Development

### Deploy

```sh
git clone https://github.com/loginov-rocks/ooyala-player-component.git
cd ooyala-player-component
npm install
```

### Run

```sh
npm start
```

### Test

```sh
npm test
```

#### Continuously

```sh
npm run test:watch
```

### Build

```sh
npm run build
```
